# 共振判定规则

本文档定义"五维信号 → 最终决策"的映射逻辑。这是整套 skill 的核心输出环节，必须严格遵守。

## 共振判定的基本原则

1. **先做硬否决**：基本面/新闻面命中任何一票否决项，直接给"排除"结论，不再走共振。
2. **基本面权重最高**：基本面看空时，其他维度看多无效。
3. **看数量也看组合**：5 维同向比 3 维同向置信度高，但"基本面+技术面+资金面"3 维比"预期面+新闻面+技术面"3 维更有价值。

## 共振组合 → 决策矩阵

| 共振组合 | 置信度 | 决策 | 仓位 |
|---------|--------|------|------|
| 5 维全看多 | ★★★★★ | 高置信买入 | 首仓 30% 目标，分批 |
| 基本面+技术面+资金面 看多，预期面/新闻面 中性 | ★★★★ | 可分批建仓 | 首仓 20%-30% |
| 基本面+技术面+资金面+预期面 看多，新闻面中性 | ★★★★ | 可分批建仓，关注催化兑现 | 首仓 30% |
| 基本面看多 + 技术面/资金面 任一看空 | ★★ | 左侧观察，不建仓 | 0% |
| 基本面中性 + 其他 3+ 维看多 | ★★ | 谨慎观察，事件驱动型可小仓位 | <10% |
| 基本面看空 | × | **排除**，不论其他维度 | 0% |
| 命中任一一票否决 | × | **排除** | 0%；持仓清仓 |
| 5 维分歧（看多看空各占）| × | 不操作，继续跟踪 | 维持现状 |

## 看多/中性/看空 判定的最小依据

不允许凭感觉给信号，每一维都要在报告里写明 ≥2 个量化依据。

**基本面看多**至少 2 条命中：
- ROE 连续 3 年 >12%
- 营收/净利增速 ≥行业均值
- 经营现金流/净利润 >1
- 估值处于历史 50% 分位以下
- 资产负债结构健康（行业适配）

**基本面看空**至少 1 条命中：
- 一票否决项
- ROE 连续 2 年下滑
- 估值 >历史 90% 分位且业绩增速跟不上

**技术面看多**：
- 月线收盘 > 20 月均线 且 均线走平或上行
- 周线 60 均线支撑有效（回踩不破）
- 日线 MACD 零轴上方金叉 或 量价配合上涨

**技术面看空**：
- 月线跌破 20 月均线超 2 个月
- 周线 MACD 顶背离
- 出现头部形态确认

**资金面看多**：
- 近 20 日北向资金净流入 >5000 万（视市值调整）
- 龙虎榜机构席位净买入
- 主力净流入连续 5 日以上

**资金面看空**：
- 北向连续大幅减持 >2 周
- 龙虎榜知名游资派发，机构净卖出
- 主力连续净流出 >10 日

**预期面看多**：
- 最新季报扣非净利超一致预期 >10%
- 近 30 日 ≥3 家券商上调目标价/评级
- 有强催化剂（强级别业绩/政策/订单）落地

**预期面看空**：
- 业绩低于一致预期 >10%
- 券商集体下调评级
- 催化剂落地后股价低位放量下跌

**新闻面看多**：
- 公司公告无异常，行业政策正向
- 大股东/管理层增持
- 公司业务推进顺利

**新闻面看空**：
- 监管问询、立案调查
- 大股东减持/质押风险高
- 业绩预告变脸

## 输出格式

报告必须包含以下"共振矩阵表"：

```
| 维度     | 信号    | 关键依据 (≥2 条) |
|---------|---------|------------------|
| 基本面   | 看多/中性/看空 | 1. xxx  2. xxx |
| 技术面   | ... | ... |
| 资金面   | ... | ... |
| 预期面   | ... | ... |
| 新闻面   | ... | ... |
```

随后给出：
- 共振强度（命中规则中哪一行）
- 最终决策
- 建议仓位
- 3 类逻辑反转触发条件
