# 仓位管理与动态跟踪规则

## 仓位规则（按持有周期适配）

### 长线（>1 年）
- **单票上限 15%**：避免单股非系统性风险
- **行业上限 30%**：规避行业系统性下行
- **首仓 ≤30% 目标仓位**：留出加仓空间
- **现金保留 ≥10%**：留出极端机会子弹

### 中线（3-6 个月）
- 单票上限 10%
- 行业上限 25%
- 设置硬性止损 -10% / -15%
- 浮盈 20% 后逐步移仓止盈线至成本

### 短线（1-3 月，事件驱动）
- 单票上限 5%
- 设置硬性止损 -8%
- 催化剂兑现后即兑现 50% 利润

## 分批建仓节奏

| 触发 | 加仓 |
|------|------|
| 首次共振信号成立 | 30% 目标仓位 |
| 回踩关键支撑（20 周线/60 日线）不破且企稳 | +30% |
| 突破前期高点 + 量能放大 | +20% |
| 季报兑现强催化 | +20% |

**禁忌**：
- 不追高加仓（突破后 5% 内不加）
- 不补跌（基本面或技术面破位后不补仓）
- 不一次性满仓

## 动态跟踪机制

### 跟踪频次

| 周期 | 跟踪内容 |
|------|---------|
| 周度 | 资金流、关键支撑/压力位是否守住、市场情绪 |
| 月度 | 基本面行业景气度、公司经营数据、估值水平 |
| 季度 | 季报数据质量、催化剂兑现度、公司治理 |

### 4 类逻辑反转触发（任一出现立即重新审视，多个同时出现立即清仓）

1. **基本面反转**
   - 行业景气度持续下行 2 季以上
   - 公司护城河弱化（毛利率连续下滑、市占率下降）
   - 季报核心数据恶化（连续 2 季营收/净利下滑、经营现金流连续 2 季为负）
   - 估值严重透支未来数年业绩

2. **技术面破位**
   - 跌破 20 月均线/60 周均线 且 无法在 3 个交易日内收复
   - 形成头部形态（M 头/头肩顶）且确认有效
   - 周线 MACD 顶背离 + 关键支撑破

3. **资金面撤离**
   - 长线资金（北向/社保）连续大幅净流出 >20 日
   - 关键支撑位未见长线资金回补
   - 高位放量派发，机构席位大额卖出

4. **催化剂证伪**
   - 业绩不及预期 + 一致预期下调
   - 政策预期落空
   - 重大利空覆盖原有利好（监管问询、行业利空政策）

## 风险事件应对

| 事件 | 处置 |
|------|------|
| 暴跌 1 日内 -7% 以上 | 暂不操作，先查公告与行业，判断是否系统性 |
| 业绩暴雷 | 立即清仓，不论买入成本 |
| 大股东巨额减持公告 | 减仓至 50% 以下，观察 1 个月 |
| 行业政策利空 | 减仓 50%，观察板块连锁反应 |
| 长期持仓累计盈利 >50% 且估值进入高分位 | 分批兑现，移仓低估值同主线标的 |
