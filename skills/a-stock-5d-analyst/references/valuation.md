# 估值方法适配指南

不同行业用错估值方法会得出极度错误的结论。本文档说明何时用哪种方法。

## 核心方法速查

| 方法 | 公式 | 适用 | 不适用 |
|------|------|------|--------|
| PE / PEG | PE = 股价 / 每股收益；PEG = PE / 净利润增速 | 盈利稳定的成熟期、成长期标的（消费、科技、医药） | 周期股、亏损股、金融地产 |
| PB / ROE | PB = 股价 / 每股净资产 | 强周期（钢铁、化工、券商）、重资产、金融地产、困境反转 | 轻资产服务业 |
| PS | PS = 总市值 / 营收 | 未盈利的成长股、SaaS、初创期 | 成熟盈利标的 |
| DCF | 折现未来自由现金流 | 现金流稳定的龙头（公用事业、高股息白马） | 业务不稳定、增长不确定 |
| EV/EBITDA | 企业价值 / 息税折旧摊销前利润 | 跨国/跨行业比较，重资产 | 服务业 |
| 股息率 | 年度分红 / 股价 | 高股息红利标的（银行、电力、运营商） | 不分红的成长股 |

## 判定的核心思路

1. **看分位优先于看绝对值**：PE=25 是高是低取决于近 5 年分位。
2. **看趋势优先于看截面**：估值在抬升还是回落更重要。
3. **看匹配度**：PEG 看 PE 与增速是否匹配，PB-ROE 看净资产与回报是否匹配。

## 常见行业适配

| 行业 | 首选 | 次选 | 说明 |
|------|------|------|------|
| 白酒、消费 | PE / PEG | DCF | 现金流稳定，业绩可预测 |
| 半导体、医药创新药 | PEG / PS | 远期 PE | 增速快，常规 PE 失真 |
| 银行 | PB / ROE | 股息率 | 资产驱动 |
| 地产 | PB / NAV | 股息率 | 资产折价 |
| 券商 | PB | PE | 业绩波动大 |
| 钢铁/化工/煤炭 | PB | EV/EBITDA | 周期股 PE 倒挂 |
| 电力、公用事业 | DCF / 股息率 | PB | 现金流稳定 |
| 港口、机场 | DCF | EV/EBITDA | 重资产长期 |
| 互联网、SaaS | PS | EV/Sales | 多数尚未稳定盈利 |
| 军工 | PEG | PS | 订单驱动 |

## 安全边际原则

- 估值合理只是入场的必要条件，不是充分条件
- 历史 20%-50% 分位是较安全的买入区间
- 历史 80% 分位以上谨慎，需要业绩高增速来消化
- 历史 5% 分位以下要警惕"价值陷阱"（便宜是有原因的）

## 估值陷阱

- **周期顶部低 PE**：钢铁/煤炭周期顶部时 PE 极低，但下个季度业绩会断崖
- **业绩支撑不可持续**：净利润主要来自一次性损益（资产出售、政府补贴）
- **会计高估**：商誉/资本化研发支出/存货跌价准备不足
